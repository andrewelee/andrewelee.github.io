Game Design

I have always been really interested in designing games. I spend a lot of time thinking about systems and paper prototyping my ideas. Most of my projects have a retro NES/SNES aesthetic, but I'd love to create a project with some nice vector art. You can check out some of my past projects above!

I started programming games back in elementary school using a program called ZZT. I later began learning Python and using the Pygame framework. Recently, I have been using the Gamemaker engine for quicker demoing and production.
